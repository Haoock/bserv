# bserv

*A Boost Based C++ HTTP JSON Server.*


## Quick Start

### Database

You can import the sample database:

- Create the database in `psql`:
  ```
  create database bserv;
  ```

- Create the table in the `shell` using a sample script:
  ```
  psql bserv < db.sql
  ```
